# My Portfolio

Project proposed in Rocketseat's Discovery course, a personal portfolio.

## Information contained
  - User profile
  - Data such
    - Location
    - Where you work
    - GitHub
    - Linkedin
    - Instagram
    - Personal website
    - Email
  - Technologies
    - Javascipt
    - Typescrypt
    - ReactJS
    - NodeJS
    - Git
    - GitHub
    - HTML
    - CSS
    - SASS
  - Experience
  - Education
